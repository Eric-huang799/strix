# Strix v2.0 - 快速开始指南

## 🚀 启动程序

### 方式1: 双击启动（推荐）
双击 `start.bat` 文件即可启动程序

### 方式2: 命令行启动
```bash
python strix.py
```

## 📖 使用步骤

1. **输入网址** - 在输入框粘贴要爬取的网页地址
2. **选择类型** - 勾选要下载的内容（图片/视频/文本）
3. **开始爬取** - 点击蓝色「开始爬取」按钮
4. **查看结果** - 点击「打开下载目录」查看下载的文件

## 📁 文件说明

- `downloads/images/` - 下载的图片
- `downloads/videos/` - 下载的视频
- `downloads/texts/` - 提取的文本
- `plugins/` - 插件目录（放自定义插件）

## 🔌 插件使用

### 已有插件
- `plugin_bilibili.py` - B站视频解析

### 编写自己的插件
1. 复制 `plugins/plugin_template.py`
2. 重命名为 `plugin_你的插件名.py`
3. 按照模板修改代码
4. 重启 Strix 即可自动加载

详细教程见 `plugins/README.md`

## 💡 提示

- 程序会自动检测已安装的插件
- 下载的视频如果是m3u8格式会自动合并
- 文本提取会自动过滤导航和广告

## ⚠️ 注意事项

- 仅爬取公开可访问的内容
- 请遵守网站的 robots.txt 规则
- 不要对网站造成过大负载
